Mileage Tracker

The mileage tracker program is made for mobile service technicians that use their personal vehicle for work and are reimbursed by their employer. The program makes it easy to keep track of trips, expenses, and profit.

This program uses a dictionary to keep track of the distance between two locations and saves and retrieves that data from a serialized file on the computer.

Database project
